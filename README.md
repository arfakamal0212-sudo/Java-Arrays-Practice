# Java Arrays Practice

This repository contains basic Java array programs with user input.

## Programs Included

1. Array Sum
2. Largest Element
3. Smallest Element
4. Reverse Array
5. Count Even Numbers
6. Count Odd Numbers
7. Array Average
8. Search Element
9. Linear Search
10. Binary Search
11. Second Largest Element
12. Second Smallest Element
13. Duplicate Elements
14. Missing Number
15. Merge Arrays

## Concepts Covered

* Array Traversal
* Array Searching
* Array Manipulation
* Missing Number Logic
* Array Merging

## Language

Java
